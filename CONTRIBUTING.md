# Contributing

Thanks for considering contributions to Nebula-Stream.

## Development Setup

1. Fork the repository.
2. Clone your fork and create a feature branch.
3. Run checks locally:

```bash
make check
cd web && npm ci && npm run build
```

## Commit Style

- Follow Conventional Commit style (`feat:`, `fix:`, `docs:`, `test:`, `refactor:`).
- Keep commits focused and small.

## Pull Requests

- Link related issues.
- Explain the motivation and design choices.
- Include test/verification notes.
